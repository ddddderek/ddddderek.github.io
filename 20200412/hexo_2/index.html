    从零开始搭建个人博客（二）
    2020/04/12 Usage Hexo Github

    通过上篇文章地讲述，相信大家已经能自由地编写、浏览自己的博客了，并且博客的 站点目录 源码也能有效地管理。本文将带你进行主题、代码高亮、文章置顶、评论等功能地配置，来使你的博客更丰富。

    主题

    打开 Hexo 官网，切换到 “Themes Tab” 下，选择你喜欢的主题，然后按照文档进行主题基础配置和功能的选配。本文我将以 A-Huhu 主题为例进行主题的配置，其他主题的配置过程也大体相同。

    安装和基础配置

    主题的安装有两种方式，一种是直接到对应的主题仓库下载压缩包，然后解压到工程根目录下的 themes 文件夹下。另一种方式是将主题fork到自己的仓库，然后 clone 添加到自己的工程中，通过 Git进行源码的管理。对比两种方式，显然第一种方式更为简单，但考虑到后期维护成本——如主题功能迭代时，我们想在我们自己的博客添加同样的功能模块等，这里我选择使用第二种方式。
